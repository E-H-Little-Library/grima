AddBibMarcFields - add a marc field to a bib record

This grima allows the addition of arbitrary MARC fields to a bib record

Input

  • MMS ID of bib record
  • New MARC fields to add formated like:
    • 505 _1 $$a asdflkj $$b word $$c Title $$1 number
    506 1_ $$a asdflkj $$b word $$b word2 $$b word3 $$c Title $$1 number

Output

  • Calls PrintBib to print the new bib record

API requirements

  • Bibs - read-write
